# ZHANG Chen Jason Homepage

Lightweight static homepage prototype for `zhangchen.info`.

## Files

- `index.html` defines the page structure.
- `styles.css` defines the responsive visual system.
- `site-data.js` stores news, research directions, themes, and publications.
- `main.js` renders dynamic lists and publication filters.

## Updating Publications

Edit `site-data.js` and add or update objects inside `publications`. The page will automatically update filters by year
and topic.

## Local Preview

Open `index.html` directly in a browser, or run any static server from this folder.
